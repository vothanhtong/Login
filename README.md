# 🧠 Neural Access - Advanced Login System

![Tech Stack](https://img.shields.io/badge/Stack-Node.js_|_Express_|_MongoDB-green?style=for-the-badge)
![Deployment](https://img.shields.io/badge/Deploy-Render-6366f1?style=for-the-badge)

**Neural Access** là một hệ thống xác thực (Login & Register) được thiết kế với phong cách tương lai, lấy cảm hứng từ các giao diện trí tuệ nhân tạo (AI) hiện đại. Hệ thống không chỉ có vẻ ngoài ấn tượng mà còn sở hữu cốt lõi mạnh mẽ với khả năng bảo mật cao và quản lý session thông minh.

## ✨ Tính năng nổi bật

- **Giao diện Neural AI**: Hiệu ứng nền với các nút mạng (neural nodes) chuyển động và ánh sáng Neon mượt mà.
- **Form thông minh**: Các trường nhập liệu có hiệu ứng Pulse, tự động kiểm soát trạng thái nhập và hiển thị lỗi tức thì.
- **Xác thực JWT**: Bảo mật thông tin người dùng bằng JSON Web Token, cấp quyền truy cập vào Dashboard riêng tư.
- **Hệ thống Quản lý**: Giao diện Dashboard tối giản sau khi đăng nhập, cung cấp cái nhìn tổng quan về tài khoản.
- **Sẵn sàng triển khai**: Tích hợp sẵn tệp `render.yaml` để đưa lên Render.com chỉ trong vài phút.

## 🛠 Công nghệ sử dụng

- **Frontend**: HTML5, CSS3 tinh khiết (Vanilla CSS), JavaScript ES6+.
- **Backend**: Node.js & Express.
- **Database**: MongoDB Atlas (Cloud Database).
- **Security**: Bcrypt (mã hóa mật khẩu) & JWT (xác thực phiên làm việc).
- **Design Inspiration**: Dự án lấy cảm hứng từ các mẫu thiết kế cao cấp trên [Colorlib](https://colorlib.com/wp/html5-and-css3-login-forms/).

## 🚀 Cài đặt và Chạy thử

1. **Clone repository:**
   ```bash
   git clone <link-repo-cua-ban>
   cd login-form
   ```

2. **Cài đặt dependencies:**
   ```bash
   npm install
   ```

3. **Cấu hình biến môi trường:**
   Tạo tệp `.env` trong thư mục gốc và nhập:
   ```env
   PORT=3000
   MONGODB_URI=your_mongodb_atlas_uri
   JWT_SECRET=your_secret_key
   ```

4. **Khởi động server:**
   ```bash
   npm start
   ```
